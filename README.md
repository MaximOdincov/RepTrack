# RepTrack
RepTrack — мобильное приложение для удобного отслеживания и анализа тренировочного прогресса. Оно помогает пользователям системно следить за своими результатами, сохранять данные о тренировках и визуализировать прогресс.

# COMING SOON

### 🔑 Основные фичи приложения:

* 📈 **Отслеживание прогресса** по упражнениям (вес, количество повторений, подходов).
* 🧩 **Шаблонизация тренировок** — создание и редактирование собственных шаблонов.
* 📊 **Статистика с графиками** — визуальное отображение изменений веса и результатов.
* ⏱️ **Таймер** для выполнения подходов.
* 📅 **Календарь** — отображение тренировок по дням и быстрый переход к любой дате.
* ☁️ **Офлайн-режим** с периодическим бэкапом данных в облако (Firebase Firestore).
* 🧍‍♂️ **Профиль пользователя** — имя, аватар, вес, статистика, прогресс.
* 🌐 **Поддержка двух языков:** русский и английский.

## ⚙️ Технологический стек

| Компонент            | Технология                                       
| -------------------- | -------------------------------------------------- 
| Язык                 | **Kotlin**                                         
| UI                   | **Jetpack Compose**                               
| Архитектура          | **Clean Architecture + MVIKotlin**                        
| DI                   | **Koin**                                
| Навигация            | **Jetpack Navigation**                             
| Асинхронность        | **Coroutines + Flow**                              
| База данных          | **Room (SQLite)**                                  
| Авторизация / Облако | **Firebase Auth + Firestore**                      
| Графики              | **MPAndroidChart / Compose Charts**                
| Тестирование         | **JUnit, MockK, coroutines-test, Compose Testing** 
| CI/CD                | **GitHub Actions**
